Ewolucja pojazdów

Dependencies

  • CMake >= 3.10
  • Qt 5.9.5 + QtQuick and QtQuick Controls
  • Boost 1.65
  • Windows: MSVC 14 (Microsoft Visual Studio Express for Dekstop 2015)

For development

  • clang-format
  • clang-tidy

Linux dependencies installation

For Ubuntu 18.04 LTS

1 sudo apt-get install <PACKAGE>

packages

1 build-essential
2 qt5-default
3 qtdeclarative5-dev
4 qml-module-qtquick-controls
5 libboost-all-dev
6 cmake

Uruchomienie

### pobranie plików

1 git clone git@github.com:kost13/cars_evolution.git

lub

1 git clone https://github.com/kost13/cars_evolution
1 cd cars_evolution
2 git submodule update --init

### budowanie

1 mkdir build && cd build
2 cmake ../ -DCMAKE_BUILD_TYPE=Release -DCMAKE_PREFIX_PATH=<QT_PATH>

dodatkowe opcja, obie sa true:

  • BUILD_TESTS
  • BUILD_GUI

np.

1 cmake -DBUILD_GUI=OFF ..

budowanie linux

1 cmake --build . --target all

budowanie windows

1 cmake --build . --target BUILD_ALL

### uruchomienie

1 ./build/src/CarsEvolution

### testy

1 ./build/tests/core_tests
2 ./build/tests/evolution_tests
3 ./build/tests/physics_tests
