crossover generates a child from the given vectors

Parameters
p1_first- first parent begin iterator
p1_last- first parent end iterator
p2_first- second parent begin iterator
p2_last- second parent end iterator
Returns
a child vector For the two given vectors it computes a new vector of the same size. The formula: new_vector[i] = w* vec1[i] + (1-w)*vec2[i] where w comes from uniform distribution U(0,1). If input vectors sizes don't match it throws std::invalid_arguments exception.
